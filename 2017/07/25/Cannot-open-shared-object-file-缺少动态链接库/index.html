缺少动态连接库.so

Cannot open shared object file: No such file or directory

总结下来主要有3种方法：

  1. 用ln将需要的so文件链接到/usr/lib或者/lib这两个默认的目录下边
  1. 修改LD_LIBRARY_PATH
  1. 修改/etc/ld.so.conf，然后刷新

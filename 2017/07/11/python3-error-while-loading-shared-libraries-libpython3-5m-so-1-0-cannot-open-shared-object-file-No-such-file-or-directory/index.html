python3: error while loading shared libraries

Posted on 2017-07-11   |   In python3

installing python3 errors raise：

python3: error while loading shared libraries: libpython3.5m.so.1.0: cannot open shared object file: No such file or directory

Error raising：

python3: error while loading shared libraries: libpython3.Xm.so.1.0: cannot open shared object file: No such file or directory

Solutions：

  1. check the dynamic lib
    ldd /usr/local/Python-3.X.X/bin/python3
  2. copy from the source code directory
    cp libpython3.Xm.so.1.0 /usr/lib/

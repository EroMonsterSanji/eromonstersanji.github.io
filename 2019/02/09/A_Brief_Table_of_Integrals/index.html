A Brief Table of Integrals

Posted on 2019-02-09 | Edited on 2019-03-04

积分总结

Basic Forms

  1. $\int k\ dx = kx + C, \quad \mathrm{(any\ number}\ k)$

  2. $\int x^n\ dx = \frac{x^{n+1}}{n+1}+C, \quad (n \ne -1)$

  3. $\int \frac{dx}{x} = \ln{|x|} + C$

  4. $\int e^x\ dx = e^x+C$

  5. $\int a^x\ dx = \frac{a^x}{\ln{a}} + C, \quad (a>0, a \ne 1)$

  6. $\int \sin{x}\ dx = -\cos{x}+C$

  7. $\int \cos{x}\ dx = \sin{x} + C$

  8. $\int \sec^2{x}\ dx = \tan{x}+C$

  9. $\int \csc^2{x}\ dx = -\cot{x} + C$

  10. $\int \sec{x} \tan{x}\ dx = \sec{x}+C$

  11. $\int \csc{x} \cot{x}\ dx = -\csc{x}+C$

  12. $\int \tan{x}\ dx = \ln{|\sec{x}|}+C$

  13. $\int \cot{x}\ dx = \ln{|\sin{x}|}+C$

  14. $\int \sinh{x}\ dx = \cosh{x}+C$

  15. $\int \cosh{x}\ dx = \sinh{x}+C$

  16. $\int \frac{dx}{\sqrt{a^2-x^2}} = \sin^{-1}\frac{x}{a}+C$

  17. $\int \frac{dx}{a^2+x^2} = \frac{1}{a} \tan^{-1} \frac{x}{a}+C$

  18. $\int \frac{dx}{x \sqrt{x^2 - a^2}} = \frac{1}{a} \sec^{-1}{\left|\frac{x}{a}\right|}$

  19. $\int \frac{dx}{\sqrt{a^2+x^2}} = \sinh^{-1}\frac{x}{a}+C, \quad (a>0)$

  20. $\int \frac{dx}{\sqrt{x^2-a^2}} = \cosh^{-1}\frac{x}{a}+C, \quad (x>a>0)$

Forms Involving $ax + b$

  1. $\int (ax+b)^n\ dx = \frac{(ax+b)^{n+1}}{a(n+1)}+C,\quad n \ne -1$

  2. $\int x(ax+b)^n\ dx = \frac{(ax+b)^{n+1}}{a^2}\left[\frac{ax+b}{n+2} - \frac{b}{n+1}\right]+C, \quad n \ne -1, -2$

  3. $\int (ax+b)^{-1}\ dx = \frac{1}{a}\ln{\left|ax+b\right|}+C$

  4. $\int x(ax+b)^{-1}\ dx = \frac{x}{a} - \frac{b}{a^2} \ln{|ax+b|}+C$

  5. $\int x(ax+b)^{-2}\ dx = \frac{1}{a^2}\left[ \ln{|ax+b|} + \frac{b}{ax+b}\right]+C$

  6. $\int \frac{dx}{x(ax+b)} = \frac{1}{b} \ln{\left|\frac{x}{ax+b}\right|}+C$

  7. $\int \left( \sqrt{ax+b} \right)^n\ dx = \frac{2}{a} \frac{\left(\sqrt{ax+b}\right)^{n+2}}{n+2}+C,\quad n\ne -2$

  8. $\int \frac{\sqrt{ax+b}}{x}\ dx = 2\sqrt{ax+b}+ b\int \frac{dx}{x\sqrt{ax+b}}$

  9. $\mathrm(a) \quad \int \frac{dx}{x \sqrt{ax+b}} = \frac{1}{\sqrt{b}} \ln{ \left| \frac{\sqrt{ax+b}-\sqrt{b}} {\sqrt{ax+b} + \sqrt{b}} \right|} + C \qquad \mathrm(b)\quad \int \frac{dx}{x \sqrt{ax-b}} = \frac{2}{\sqrt{b}} \tan^{-1} \sqrt{\frac{ax-b}{b}}+C$

  10. $\int \frac{\sqrt{ax+b}}{x^2}\ dx = -\frac{\sqrt{ax+b}}{x}+\frac{a}{2}\int \frac{dx}{x \sqrt{ax+b}}+C$

  11. $\frac{dx}{x^2\sqrt{ax+b}} = -\frac{\sqrt{ax+b}}{bx} - \frac{a}{2b} \int \frac{dx}{x \sqrt{ax+b}}+C$

Forms Involving $a^2 + x^2$

  1. $\int \frac{dx}{a^2+x^2} = \frac{1}{a} \tan^{-1} \frac{x}{a}+C$

  2. $\int \frac{dx}{ {(a^2 + x^2)}^2 } = \frac{x}{2 a^2 (a^2 + x^2)} + \frac{1}{2 a^3} \tan^{-1} {\frac{x}{a}} + C$

  3. $\int \frac{dx}{\sqrt{a^2+x^2}} = \sinh^{-1} \frac{x}{a}+C = \ln{\left(x+ \sqrt{a^2+x^2}\right)+C}$

  4. $\int \sqrt{a^2+x^2}\ dx = \frac{x}{2}\sqrt{a^2+x^2} + \frac{a^2}{2} \ln{\left(x + \sqrt{a^2+x^2}\right)}+C$

  5. $\int x^2 \sqrt{a^2+x^2}\ dx = \frac{x}{8} \left(a^2 + 2x^2\right) \sqrt{a^2+x^2} - \frac{a^4}{8} \ln \left(x + \sqrt{a^2+x^2}\right)+C$

  6. $\int \frac{\sqrt{a^2+x^2}}{x}\ dx = \sqrt{a^2+x^2} - a \ln{\left|\frac{a+ \sqrt{a^2+x^2}}{x}\right|}+C$

  7. $\int \frac{\sqrt{a^2+x^2}}{x^2}\ dx = \ln{\left(x + \sqrt{a^2 + x^2}\right) - \frac{\sqrt{a^2+x^2}}{x}+C}$

  8. $\int \frac{x^2}{\sqrt{a^2+x^2}}\ dx = -\frac{a^2}{2} \ln{\left(x + \sqrt{a^2+x^2}\right)} + \frac{x\sqrt{a^2+x^2}}{2}+C$

  9. $\int \frac{dx}{x \sqrt{a^2+x^2}} = -\frac{1}{a} \ln{\left|\frac{a + \sqrt{a^2+x^2}}{x}\right|+C}$$

  10. $\int \frac{dx}{x^2 \sqrt{a^2+x^2}} = - \frac{\sqrt{a^2+x^2}}{a^2x}+C$

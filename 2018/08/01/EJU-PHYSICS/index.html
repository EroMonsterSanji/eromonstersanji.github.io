运动学

动量，冲量

动量变化量为冲量 $ m \overrightarrow{v_2} - m \overrightarrow{v_1} = \overrightarrow{F} \Delta t $

动量守恒定律

物体为两个以上，并且受到外界冲量为0时，动量总和保持不变。

动量守恒，动能守恒应用范围

如果只有重力和系统内弹簧弹力做功，系统的机械能守恒；如果系统所受合外力为零，则系统动量守恒。

反発係数

$v_1$, $v_2$为冲突前两物体速度，$v_1’$, $v_2’$为冲突后的速度。
$e$为冲突后速度差与冲突前速度差的比值的相反数。

  • $e = 1$
    弹性碰撞
  • $e \ne 1$
    非弹性碰撞
  • $e = 0$
    完全非弹性碰撞

弹性碰撞后的速度关系

当发生弹性碰撞时，系统动能保持不变（动量始终守恒）。

单振动

单振动的特征

单振动轨迹为匀速圆周运动的正投影。

$ T = \frac{2 \pi}{\omega} \quad f = \frac{1}{T} $

  • 速度：

    • 振动中心最大：$ v_{max} = A \omega $

    • 两端：$ v = 0 $

  • 加速度：

    • 两端最大：$ a_{max} = A {\omega}^2 $

    • 振动中心：$ a = 0 $

加速度与位置坐标的关系：$ a = - {\omega}^2 x $

気体热力学

気体の圧力

由于收到热运动气体分子的撞击，容器面收到来自气体的 压强。单位为 $Pa$。

理想气体定律

气体 压强 的大小与气体 物质的量 ( $n$ )、气体的 温度 ( $T$ ) 成正比，与气体的 体积 ( $V$ ) 成反比，$R$ 为 通用气体常量：$8.31 J/(mol \cdot K)$

分子の平均運動エネルギー

$k_B$ 为玻尔兹曼常数，$k_B = \frac{R}{N_A} = 1.38064852(79) \times 10^{-23} J / K$，其中 $R$ 为气体常数。

推导过程

边长为 $L$ 的立方体理想容器中，气体分子与内壁发生弹性碰撞。
气体分子收到冲量为：

在时间 $t$，运动了 $v_xt$ 的距离。每往复 $2L$ 与内壁 $S$ 碰撞一次。因此碰撞次数为：$v_xt / 2L$
冲量合计：

单一分子冲量合计的平均值的 $N$ 倍即为全部分子对 $S$ 给的冲量 $Ft$。

因为 $v^2 = {v_x}^2 + {v_y}^2 + {v_z}^2$，
所以 $\overline{v^2} = \overline{ {v_x}^2} + \overline{ {v_y}^2} + \overline{ {v_z}^2}$
又因 $\overline{ {v_x}^2} = \overline{ {v_y}^2} = \overline{ {v_z}^2}$，
所以 $\overline{v^2} = 3 \overline{ {v_x}^2}$

结合 $F = \frac{N m \overline{ {v_x}^2}}{L}$,

将结果与状态方程式 $PV = nRT = \frac{N}{N_A}RT$ 对比：

気体の内部エネルギー

内部エネルギー $U$ 为分子动能总和。

内部エネルギー $U$ 与绝对温度 $T$ 成正比。

气体做功

定积变化，微小变化的场合：$W = P \Delta V = n R \Delta T$

一般情况：$W = \int_{V_A}^{V_B} P\ dV$

热力学第1法则

特殊变化

  • 等温变化：$ \Delta U = 0 $

  • 断热变化：$ Q = 0 $

  • 定积变化：$ W = 0 $

  • 定压变化：$ W = - P \Delta V $

气体比热

  • 定积变化：$ Q_v = n C_v \Delta T $

  • 定压变化：$ Q_p = n C_p \Delta T $

$ \Delta U = n C_v \Delta T $ 对任意的变化成立，即 $ \Delta U = \frac{3}{2} n R \Delta T $

对于定积变化，$Q = \Delta U$，因此成立。

断热变化

断热压缩 $\Longrightarrow$ 温度上升 $\quad \quad \quad$ 断热膨胀 $\Longrightarrow$ 温度下降
因断热时，$Q = 0$，$\Delta U = W$。

$ \gamma $ 为 比热比。

对于单分子理想气体：$\gamma = \frac{\frac{5}{2} R} {\frac{3}{2} R} = \frac{5}{3}$

气体混合

两个容器加入不同物质的量，温度，压强的气体。打开阀门将其混合。

混合前后：

  • 物质的量总和不变
  • 两侧压强相等

气体混合的三种状况

  1. 由外部调节气体温度 $\Rightarrow$ 联立彼此状态方程式
  2. 断热容器混合 $\Rightarrow$ 内部エネルギー总和不变
  3. 一边为真空，断热容器中的扩散 $\Rightarrow$ 温度不变

基尔霍夫定律

  • 基本术语

    • 支路(branch)
      单个或多个元件串联
    • 节点(node)
      三条及以上支路的连接点
    • 回路(loop)
      闭合路径
    • 网孔(mesh)
      内部不包含支路的回路

  • 基本关系

    网孔 = 支路 - 节点 + 1

  • 基尔霍夫电流定律 ($KCL$)
    任何电路，任何节点，任意时刻，流入、流出该节点的电流之和恒等于0

    本质：电荷守恒定律

    可应用广义节点/超节点

  • 基尔霍夫电压定律 ($KVL$)
    任何电路，任何回路，任意时刻，沿回路方向，所有元件电压降之和为0

    本质：能量守恒定律

惠斯登電橋

待测电阻 $R_x$ ，和已知電阻的可变电阻器 $R_2$ 、电阻 $R_1$ 和电阻 $R_3$ 。

在一个电路内，将 $R_1$ 和 $R_2$ 串联，$R_3$ 和 $R_x$ 串联，再將这兩个串联的電路并联，在 $R_1$ 和 $R_2$ 之间的电线中点跟在 $R_3$ 和 $R_x$ 之間的電線中接点驳上一条电线，在这条电线上放置檢流計。

當 $ R_2 / R_1 = R_x / R_3 $ 时，电桥平衡，检流计无电流通过。由于是否有电流经过是十分敏感的，惠斯登桥可以获取颇精确的测量。

  • 半径为 $r$ 的环形导线，通过的电流为 $I$，环形中心产生 $H_1[A/m]$ 的磁场

  • 电流为 $I$ 的直导线，距离导线距离为 $r$ 的位置的磁场强度为 $H_2[A/m]$

  • 电流为 $I$，匝数为 $n$ 的螺线圈（ソレノイド）中的磁场强度$H_3[A/m]$

方向符合右手螺旋定则

磁通量 $\Phi_B$，磁感应强度，磁通密度 $B$

磁通量 $\Phi_B$，是通過某给定曲面的磁場（亦称为磁通量密度）的大小的度量。磁通量的国际单位制單位是韦伯。

磁感应强度也被称为磁通量密度或磁通密度，是一个表示贯穿一个标准面积的磁通量的物理量，其符号是 $B$，國際單位制導出單位是 $T$。

電流が磁場から受ける力、ローレンツ力

在磁感应强度 $B [Wb/m^2]$ 的磁场中，放置电流为 $I$ 的长度为 $l$ 的导线，其受到的力的大小为 $F$。同时，在相同磁场中以速度 $v$ 运动的点电荷（电荷量为 $q$），收到的洛伦兹力的大小为 $f$。

方向符合左手定则

带电粒子在磁场中运动

粒子在磁场中收到洛伦兹力。

洛伦兹力公式与向心力公式联立可得 运动半径 $r = \frac{mv}{Bq}$

完成整个圆周运动的周期为 $T = \frac{2m \pi}{Bq}$

ファラデーの電磁誘導の法則

任何封闭电路中感应电动势(誘導起電力) $V$ 的大小，等于穿过这一电路的磁通量(磁束)的变化率。

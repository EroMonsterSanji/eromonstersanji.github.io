直线方程式，2直线关系

直线方程式

  1. 斜率$m$，截距$n$：
    $y = mx + n$
  2. 通过点 $(x_1,\ y_1)$：
    • 斜率$m$：
      $y-y_1 = m(x-x_1)$
    • 与x轴垂直：
      $x = x_1$
  3. 通过 $(x_1,\ y_1),\ (x_2,\ y_2)$：
    • $x_1 \ne x_2$:
      $y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1)$
    • $x_1 = x_2$
      $x = x_1$
  4. 通过 $(a,\ 0),\ (0,\ b)\ [a \ne 0,\ b \ne 0]$
    $\frac{x}{a} + \frac{y}{b} = 1$

复数运算

  1. 共轭复数
    $a + bi$ 与 $a - bi$ 互为共轭复数
    其和与积均为实数
  2. 四则运算
    1. $(a + bi) + (c + di) = (a+c) + (b+d)i$
    2. $(a + bi) - (c + di) = (a-c) + (b-d)i$
    3. $(a + bi) * (c + di) = (ac - bd) + (ad + bc)i$
    4. $\frac{(a + bi)}{(c + di)} = \frac{ac+bd}{a^2+b^2} + \frac{ad-bc}{a^2+b^2}i$

二次函数解的公式

二次方程 $ax^2 + bx + c = 0$ 的解为 $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

特别地，对于 $b = 2b’$ ， $x = \frac{-b’ \pm \sqrt{b’^2 - ac}}{a}$

三角関数

三角関数の基本公式

  1. $ cos^2 \theta + sin^2 \theta = 1 $

  2. $ 1 + tan^2 \theta = \frac{1}{cos^2 \theta} $

  3. $ \tan{\theta} = \frac{\sin{\theta}}{\cos{\theta}} $

加法定理、三角関数の合成

  • 加法定理

    • $ \sin{(\alpha + \beta)} = \sin{\alpha} \cos{\beta} + \cos{\alpha} \sin{\beta} $

    • $ \sin{(\alpha - \beta)} = \sin{\alpha} \cos{\beta} - \cos{\alpha} \sin{\beta} $

    • $ \cos{(\alpha + \beta)} = \cos{\alpha} \cos{\beta} - \sin{\alpha} \sin{\beta} $

    • $ \cos{(\alpha - \beta)} = \cos{\alpha} \cos{\beta} + \sin{\alpha} \sin{\beta} $

    • $ \tan{(\alpha + \beta)} = \frac{\tan{\alpha} + \tan{\beta}}{1 - \tan{\alpha} \tan{\beta}} $

    • $ \tan{(\alpha - \beta)} = \frac{\tan{\alpha} - \tan{\beta}}{1 + \tan{\alpha} \tan{\beta}} $

  • 2倍角の公式

    • $ \sin{2 \alpha} = 2 \sin{\alpha} \cos{\alpha} $

    • $ \cos{2 \alpha} = cos^2 \alpha - sin^2 \alpha = 2 cos^2 \alpha - 1 = 1 - 2 sin^2 \alpha $

    • $ \tan{2 \alpha} = \frac{2 \tan{\alpha}}{1 - tan^2 \alpha} $

  • 半角の公式

    • $ sin^2 \frac{\alpha}{2} = \frac{1-cos\alpha}{2} $

    • $ cos^2 \frac{\alpha}{2} = \frac{1+cos\alpha}{2} $

    • $ tan^2 \frac{\alpha}{2} = \frac{1-cos\alpha}{1+cos\alpha} $

  • 三角関数の合成

    • $ a \sin{\theta} + b \cos{\theta} = r \sin{\theta + \alpha} = r \cos{\theta - \beta} $

      $ r = \sqrt{\alpha^2 + \beta^2},\ (a,\ b) = (r\cos{\alpha},\ r\sin{\alpha}),\ (b,\ a) = (r\cos{\beta},\ r\sin{\beta}) $

常用导数公式

  1. $C’ = 0$

  2. $(x^a)’ = ax^{a-1}$

  3. $(a^x)’ = a^x \cdot \ln{a} \quad (a > 0, a \ne 1)$

  4. $(e^x)’ = e^x$

  5. $(\log_a{|x|})’ = \frac{1}{x \ln{a}} \quad (\ln{|x|})’ = \frac{1}{x}$

  6. $(\sin{x})’ = \cos{x}$

  7. $(\cos{x})’ = -\sin{x}$

  8. $(\tan{x})’ = \sec^2{x}$

  9. $(\cot{x})’ = -\csc^2{x}$

  10. $(\sec{x})’ = \sec{x} \cdot \tan{x}$

  11. $(\csc{x})’ = - \csc{x} \cdot \cot{x}$

  12. $(\arcsin{x})’ = \frac{1}{\sqrt{1-x^2}}$

  13. $(\arccos{x})’ = - \frac{1}{\sqrt{1-x^2}}$

  14. $(\arctan{x})’ = \frac{1}{\sqrt{1+x^2}}$

  15. $(\mathrm{arccot} \space x)’ = - \frac{1}{1+x^2}$

基本求导法则

  1. 线性法则： $(au + bv)’ = au’ + bv’$

  2. 积法则： $(uv)’ = uv’ + u’v$

  3. 商法则： $(\frac{u}{v})’ = \frac{u’v - uv’}{v^2},\ v \ne 0$

  4. 链式法则： $(f[u(x)])’ = f’[u(x)] \cdot u’(x)$

  5. 反函数法则： $f’(x) = \frac{1}{\varphi’(y)}, \varphi’(y) \ne 0$,\ $\varphi$是$f$的反函数

式与曲线

二次曲线

放物線

顶点为原点，焦点为 $(p,\ 0)$，准线为 直线 $x = -p$

椭圆

  • 中心为原点，长轴 $2a$，短轴 $2b$
  • 焦点 $F(c,\ 0),\ F’(-c,\ 0) \quad c = sqrt{a^2 - b^2}$
  • 椭圆上任意一点到两焦点距离和为 $2a$

同时适用于焦点在 $x,\ y$轴

双曲线

  1. $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1\ (a > 0,\ b > 0)$

    • 中点为原点，顶点为 $(a,\ 0),\ (-a,\ 0)$
    • 焦点 $F(c,\ 0),\ F’(-c,\ 0) \quad c = sqrt{a^2 + b^2}$
    • 渐近线：$\frac{x}{a} - \frac{y}{b} = 0,\ \frac{x}{a} + \frac{y}{b} = 0$
    • 双曲线上任意一点到焦点距离之差为 $2b$

  2. $\frac{x^2}{a^2} - \frac{y^2}{b^2} = -1\ (a > 0,\ b > 0)$

    • 中点为原点，顶点为 $(0,\ b),\ (0,\ -b)$
    • 焦点 $F(0,\ c),\ F’(0,\ -c) \quad c = sqrt{a^2 + b^2}$
    • 渐近线：$\frac{x}{a} - \frac{y}{b} = 0,\ \frac{x}{a} + \frac{y}{b} = 0$
    • 双曲线上任意一点到焦点距离之差为 $2b$
